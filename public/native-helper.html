Native FFmpeg Helper Setup

The native helper is optional. Detection and direct browser downloads work without it; local FFmpeg export, original trim, generated previews, and thumbnails use it when ready.

Windows Beta Setup

Run the PowerShell setup wrapper from the project or beta setup package. It checks Node.js, FFmpeg, and FFprobe first, then asks before installing missing dependencies with winget. 

powershell -ExecutionPolicy Bypass -File .\native\ffmpeg-helper\scripts\setup-windows.ps1 -ExtensionId <your-extension-id>

Requirements

  • Node.js 20 or newer.
  • FFmpeg and FFprobe available on PATH.
  • A C# compiler, such as .NET Framework Developer Pack or Visual Studio Build Tools.

Troubleshooting

  • NATIVE_UNAVAILABLE: install the helper with the current extension ID, then restart Chrome.
  • FFMPEG_NOT_FOUND: install FFmpeg/FFprobe and open a fresh terminal before rerunning setup.
